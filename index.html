About

flipsnap.js is JavaScript library. It can implement flip and slid UI easily.

Support browser

Mobile

  • iOS Safari (iOS4+)
  • Android Browser (Android 2.1+) (*1)
  • Android Firefox Mobile 9.0+
  • Android Opera Mobile 11.50+
  • Window8 IE10+

PC

  • IE9+
  • Google Chrome
  • Opera
  • Firefox
  • Safari

*1: Android2.1 device dependence, touchmove event is not fire until the touchend event
