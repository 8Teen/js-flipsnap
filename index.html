About

flipsnap.jsはスマートフォン向けのスワイプしてスライドするUIを簡単につくることができるJavaScriptのライブラリです。スマートフォン向けのライブラリですがPCでも動作します（IEは9以上）。

Supoort browser

Mobile

  • iOS Safari (iOS4+)
  • Android Browser (Android 2.1+) (*1)
  • Android Firefox Mobile 9.0+
  • Android Opera Mobile 11.50+
  • Windows Phone7 IE9+ (*2)

PC

  • IE9+
  • Google Chrome
  • Opera
  • Firefox
  • Safari

*1: Androidは（特に2.1などで）機種によってはtouchmoveのイベントがtouchendまで発生しないものあるのでタッチコントロールがうまく動かない端末もある（参考記事）。

*2: Windows Phone7のIE9はmousemoveがうまく発火せずにタッチコントロールは動作せず（moveToやtoNext等のメソッドは動く）。
